# Post-MVP Feature Ideas

Ideas to revisit after core implementation is complete. Ranked by impact/effort ratio.

## WOW Factor Candidates

### 1. Risk Score Widget
- Pure calculation based on response patterns, document completeness, time to deadline
- Visual gauge or percentage display
- **Why it's good:** High visual impact, no API calls, shows domain thinking
- **Effort:** Low (~2-3 hours)

### 2. Mini AI Chat
- "Ask about this client" interface with tool calling
- Multi-turn conversation about client status
- **Why it's good:** Shows advanced AI SDK usage (tool calling)
- **Effort:** Medium-High (~4-6 hours)

### 3. Quick Actions Bar
- Context-sensitive buttons based on status
- Red: "Send personal text", "Schedule call", "Flag for review"
- Amber: "Review documents", "Send reminder"
- Green: "Send thank you", "Update notes"
- **Why it's good:** Practical UX, shows workflow understanding
- **Effort:** Low-Medium (~2-3 hours)

### 4. Sentiment Timeline
- Chart showing client mood over time from conversations
- Visual trend indicator
- **Why it's good:** Unique visualization, data-driven
- **Effort:** Medium (requires charting library)

## Other Ideas

### Keyboard Shortcuts
- `cmd+k` command palette for power users
- Arrow key navigation in contact list

### Smart Escalation Timing
- "Best time to contact: Tuesday 2pm" based on response patterns
- Simple calculation, impressive presentation
